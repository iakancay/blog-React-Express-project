## Packages and Libraries
 ### Client Side
  - React
  - Material UI (for ui design)
  - Yup and react-hook-form (for form validation)
  - FileBase64 (for uploading file)
  - react-router-dom@v6 (for routing)

 ### Server Side
   - Express
   - cors
   - mongoose
